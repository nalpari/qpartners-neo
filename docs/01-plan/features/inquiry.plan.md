# 문의 등록 Planning Document

> **Summary**: 로그인 전/후 문의 등록 + 문의유형별 수신 담당자 메일 발송
>
> **Project**: qpartners-neo
> **Author**: CK
> **Date**: 2026-03-28
> **Status**: Draft
> **화면설계서**: p.42-43 (confirmed)

---

## 1. Overview

HWJ(한화재팬)에 문의 메일을 발송하는 기능. 로그인 전/후 모두 접근 가능하며, 문의유형에 따라 수신 담당자가 차별화됨. 문의유형별 수신 담당자 이메일은 공통코드로 관리.

---

## 2. Scope

### 2.1 In Scope
- [ ] 문의 등록 API (로그인 전: 직접 입력, 로그인 후: 사용자 정보 자동)
- [ ] 문의유형별 수신 담당자에게 메일 발송
- [ ] 작성자에게 접수 확인 메일 발송
- [ ] 문의 내역 DB 저장

### 2.2 Out of Scope
- 문의 등록 UI (프론트 담당)

---

## 3. Requirements

| ID | Requirement | Priority | 화면설계서 |
|----|-------------|----------|-----------|
| FR-01 | 로그인 전: 회사명, 성명, 전화번호, 이메일 직접 입력 | High | p.43 #1 |
| FR-02 | 로그인 후: 로그인한 사용자 정보 자동 표시 | High | p.42 #1 |
| FR-03 | 문의유형 선택 (8가지) | High | p.42 #6 |
| FR-04 | 제목, 내용 입력 | High | p.42 #2, #3 |
| FR-05 | 문의유형별 수신 담당자에게 메일 발송 (최대 3명) | High | p.42 |
| FR-06 | 접수 Alert "문의가 접수되었습니다" + 화면 초기화 | Medium | p.42 #5 |

---

## 4. API Endpoints

```
POST /api/inquiries    → 문의 등록 + 메일 발송
```

---

## 5. 문의유형 (공통코드 관리)

| 유형 | 일본어 | 한국어 |
|------|--------|--------|
| 01 | ログインIDおよびログインに関するお問い合わせ | 로그인ID 및 로그인에 관한 문의 |
| 02 | 会員登録に関するお問い合わせ | 회원등록에 관한 문의 |
| 03 | 施工ID/施工研修に関するお問い合わせ | 시공ID 및 시공연수에 관한 문의 |
| 04 | 配信メールに関するお問い合わせ（受信拒否を含む） | 뉴스레터에 관한 문의 (수신거부 포함) |
| 05 | 営業・マーケティング資料に関するお問い合わせ | 영업, 마케팅 자료에 관한 문의 |
| 06 | 技術資料に関するお問い合わせ | 기술자료에 관한 문의 |
| 07 | 品質保証資料に関するお問い合わせ | 품질보증자료에 관한 문의 |
| 08 | 退会に関するお問い合わせ | 회원탈퇴에 관한 문의 |

수신 담당자 이메일: 공통코드 `relCode1~3`으로 관리 (최대 3개)
